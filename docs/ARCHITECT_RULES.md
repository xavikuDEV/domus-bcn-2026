# 📏 ARCHITECT RULES: Domus BCN 2026

## 💻 Desarrollo
- **Framework:** Next.js 15 con App Router.
- **Componentes:** Usar Server Components por defecto. Solo usar 'use client' si hay interactividad.
- **Tipado:** TypeScript estricto. Prohibido usar `any`.

## 🤖 Agentes
- **Skills:** Cada nueva habilidad de agente debe documentarse en `.antigravity/skills/`.
- **Logs:** Toda acción crítica debe registrarse en la Bitácora de Notion.
